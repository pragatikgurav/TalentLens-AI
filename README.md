# 🎯 TalentLens AI

An intelligent, NLP-powered applicant tracking and resume screening system that automatically parses candidate resumes, extracts core features, and ranks them against job descriptions with deep analytical insights.

🔗 **[Live Demo](https://pragatikgurav.github.io/TalentLens-AI/)**

---

## 🚀 Overview

TalentLens AI simplifies the recruitment workflow by leveraging Natural Language Processing (NLP) to break down complex resumes into structured data. It compares candidate profiles against specific job descriptions using advanced similarity metrics, providing recruiters with an instant, data-driven match analysis.

### Key Features
* **Automated Resume Parsing:** Extracts skills, education, experience, and certificates automatically.
* **Semantic Match Scoring:** Uses NLP metrics to evaluate how well a candidate fits a job description beyond simple keyword matching.
* **Comprehensive Candidate Insights:** Generates radar charts for skill gaps, strengths, and AI-driven recommendations.
* **Intuitive Recruitment Dashboard:** Easily manage job descriptions, upload resumes, and track candidate statuses.
